
# Contributing

Этот репозиторий — живой. Можно предлагать правки и темы через issues / PR.

## Что приветствуется

- правки по структуре и навигации
- улучшение шаблонов/чеклистов/DoD
- предложения тем из реальных кейсов
- исправления опечаток и формулировок

## Что не приветствуется

- “как в книжке” без практической ценности
- холивары про инструменты ради инструментов
- раскрытие внутренней информации / названий систем / конфиденциальных деталей

## Соглашения по структуре

- статьи: `articles/YYYY-MM/<slug>.md`
- артефакты: `artifacts/<type>/<name>.md`
- ассеты: `assets/<slug>/...`

Если сомневаешься — создай issue.
